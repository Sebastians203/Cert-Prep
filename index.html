Domain 1

The fundamentals of HTML

The four areas of HTML are the DOCTYPE, HTML, Head, and Body.

  • DOCTYPE - Indicates the type of document
  • HTML - Opens and closes the document's code
  • Head - contains titles, links, and tags that DO NOT show on the webpage
  • Body - contains elements that DO show on the webpage

Domain 2

Cascading Style Sheets

The three types of CSS style sheets are inline styles, internal styles, and external style sheets linkeed in the document head tag.

Rules in CSS consists of tags, classes, and IDs. They have properties such as "width". These are seperated by semi colons.


Domain 3

Headings, Line breaks, and Lists

Some common tags in HTML are the heading tags which have different sized text in descending order.

Different List tags like the unordered list, the ordered lists, and list item are used for creating lists.

The BR tag is used to create line breaks. P tags are used to create normal text.


Domain 4

Images, Videos, Audio Tags

To link an Image to a page, the dimensions must be correct for it to fit in the page.

CSS is used to edit image layouts on pages. Hyperlinks are used to link images into webpages.

While linking a video to a page, control and caption tags can be added to improve the webpage.


Domain 5

CSS Positioning

While editing CSS style sheets some options for positioning are, changing max width or max length.

Float positioning is used to move an item to a specific area. Rekative positioning moves elements offset without changing layout flow. Absoulte Positioning places an element in an exact location.
